- [Requirement 1] The robot must start and end its tour at depot city 0.
- [Requirement 2] The robot must visit all cities exactly once, excluding the depot city revisit.
- [Requirement 3] The travel cost between cities is calculated using the Euclidean distance.
- [Requirement 4] The objective is to find the shortest possible tour.
- [Requirement 5] The solution should be derived using the Lin-Kernighan algorithm for TSP.
- [Requirement 6] The output must be formatted to include the tour as a list of city indices starting and ending with the depot city, followed by the total travel cost.
