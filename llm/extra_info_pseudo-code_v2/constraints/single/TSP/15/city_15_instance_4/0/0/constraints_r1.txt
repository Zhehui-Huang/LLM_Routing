- [Requirement 1] The robot must start at the depot city 0 and return to the depot city 0 after visiting all other cities.
- [Requirement 2] Each city, except the depot city, must be visited exactly once by the robot.
- [Requirement 3] Travel cost between any two cities is calculated using the Euclidean distance formula.
- [Requirement 4] The total tour should be the shortest possible, considering the Euclidean travel distances.
- [Requirement 5] Output should include the tour as a list of city indices and also the total travel cost of the tour. The list should start and end at the depot city 0.
