- [Requirement 1] The robot must start and end the tour at depot city 0.
- [Requirement 2] Each city must be visited exactly once by the robot.
- [Requirement 3] Output format should include the tour as a list of city indices starting and ending at the depot city.
- [Requirement 4] Output format should include the total travel cost of the tour.
- [Requirement 5] Output format should include the maximum distance between any two consecutive cities in the tour.
- [Requirement 6] The goal is to minimize the longest distance between any two consecutive cities in the tour.
