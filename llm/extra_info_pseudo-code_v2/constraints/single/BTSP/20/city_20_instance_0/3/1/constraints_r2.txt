- [Requirement 1] The robot must start and end at depot city 0.
- [Requirement 2] The robot must visit each city exactly once in between.
- [Requirement 3] The robot's movement cost is calculated using the Euclidean distance between cities.
- [Requirement 4] Minimize the longest distance between any two consecutive cities in the tour.
- [Requirement 5] Output should include the tour as a list of city indices starting and ending at city 0.
- [Requirement 6] Output should include the total travel cost of the tour.
- [Requirement 7] Output should include the maximum distance between any two consecutive cities in the tour.
