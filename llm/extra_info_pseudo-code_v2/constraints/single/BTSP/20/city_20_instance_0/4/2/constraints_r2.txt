- [Requirement 1] The robot must start and end its tour at the depot city 0.
- [Requirement 2] The robot must visit each city exactly once.
- [Requirement 3] The objective is to minimize the longest distance between any two consecutive cities in the robot's tour.
- [Requirement 4] The travel cost between any two cities is calculated using the Euclidean distance.
- [Requirement 5] The output must include the tour as a list of city indices, starting and ending with the depot city.
- [Requirement 6] The output must include the total travel cost of the tour.
- [Requirement 7] The output must include the maximum distance between any two consecutive cities in the tour.
