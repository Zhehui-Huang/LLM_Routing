- [Requirement 1] The robot needs to start and end its tour at the depot city, which is city 0.
- [Requirement 2] The robot must visit exactly 4 cities in total, including the depot city.
- [Requirement 3] Compute the total travel cost using the Euclidean distance between consecutive cities in the tour.
- [Requirement 4] The optimizer should use the General Variable Neighborhood Search (GVNS) algorithm to search for the shortest possible tour.
- [Requirement 5] Output the tour as a list of city indices, beginning and ending at depot city 0.
- [Requirement 6] Output the total travel cost of the tour.
