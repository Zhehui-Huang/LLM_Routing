- [Requirement 1] The robot must start and end at the depot city (index 0).
- [Requirement 2] Each city must be visited exactly once, except for the depot city which is visited twice (start and end).
- [Requirement 3] The robot can travel directly between any two cities.
- [Requirement 4] The travel cost between cities is calculated using the Euclidean distance.
- [Requirement 5] The objective is to find the shortest possible tour that satisfies the above conditions.
