- [Requirement 2] The robot must visit each city exactly once, except the depot city.
- [Requirement 3] The travel cost is calculated as the Euclidean distance between cities.
- [Requirement 4] A single tour must include all the cities exactly once, returning to the starting point.

- [Requirement 6] Output the total travel cost of the tour.
