- [Requirement 1] There are 22 cities, including a depot city. Each city, except the depot, must be visited exactly once by the robots.
- [Requirement 2] All robots start and finish at the depot city.
- [Requirement 3] The primary objective is to minimize the maximum distance traveled by any single robot.
- [Requirement 4] Each tour's travel cost is calculated as the Euclidean distance between consecutive cities.
- [Requirement 5] Output the tours for each robot as a list of city indices, starting and ending at the depot.
- [Requirement 6] Output the travel cost for each tour.
- [Requirement 7] Output the maximum travel cost among all tours.
