- [Requirement 1] There is one robot starting at depot city 0.
- [Requirement 2] The robot must visit each of the 20 cities exactly once and return to the depot city.
- [Requirement 3] Travel cost is calculated as the Euclidean distance between the cities.
- [Requirement 4] The output should be a list indicating the tour starting and ending at the depot city 0.
- [Requirement 5] The output should also include the total travel cost of the tour.
- [Requirement 6] Use the Christofides algorithm to find the approximate shortest tour.
