- [Requirement 1] The robot must visit exactly 8 cities, including the depot city.
- [Requirement 2] The tour must start and end at the depot city (city 0).
- [Requirement 3] Travel cost is calculated using the Euclidean distance between cities.
- [Requirement 4] The robot can travel between any two cities.
- [Requirement 5] The objective is to find the shortest possible tour satisfying the above conditions.
- [Requirement 6] The output must list the tour as indices of cities, starting and ending at the depot city.
- [Requirement 7] The output must also include the total travel cost of the tour.
- [Requirement 8] Utilize the General Variable Neighborhood Search (GVNS) algorithm to solve the problem.
