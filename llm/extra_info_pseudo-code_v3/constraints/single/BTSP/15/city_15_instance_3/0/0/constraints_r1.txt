- [Requirement 1] The robot must visit each city exactly once and return to the starting depot city 0.
- [Requirement 2] The robot travels in a complete graph with Euclidean distances between cities representing travel costs.
- [Requirement 3] Minimize the longest distance between any two consecutive cities in the tour.
- [Requirement 4] Output the tour as a list of city indices starting and ending at depot city 0.
- [Requirement 5] Output the total travel cost of the tour.
- [Requirement 6] Output the maximum distance between any two consecutive cities in the tour.
