- [Requirement 1] The robot must visit each city exactly once and return to the depot city, forming a closed loop.
- [Requirement 2] The goal is to minimize the longest distance between any two consecutive cities in the tour.
- [Requirement 3] The output should include the tour as a list of city indices starting and ending at the depot city (0).
- [Requirement 4] The output should also include the total travel cost of the tour.
- [Requirement 5] The output should include the maximum distance between any two consecutive cities in the tour.
